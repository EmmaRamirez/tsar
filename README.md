## Installation

```
npm install @emmaramirez/tsar --save
```

## Usage

```
import { <utility-name> } from '@emmaramirez/tsar'
```

tsar provies these typed utilities

- Nullable
- Partial
- Readonly
- Queue
- Action
- BaseBinaryTree
- BinaryTreeNode
- Maybe
